Video Downloader - Instructions
==============================

1. Make sure ffmpeg is installed:
   - Download from: https://www.gyan.dev/ffmpeg/builds/
   - Extract ffmpeg.exe to this folder or add to PATH

2. Run VideoDownloader.exe

3. The application includes:
   - VideoDownloader.exe - Main application
   - yt-dlp.exe - Video download engine

4. First time setup:
   - Go to Settings tab
   - Enter credentials if needed for private videos
   - Save settings

5. To download a video:
   - Paste the video URL
   - Select quality
   - Click Download Video

Note: Downloaded videos will be saved to the folder you specify.
